# Simple Banking Application

A console-based Java application that simulates basic banking operations using object-oriented programming.

## Features
- Deposit money into the account
- Withdraw money from the account with balance check
- Check current account balance
- Simple and interactive console interface

## Technologies
- Java (OOP concepts: classes, methods, encapsulation)
- Scanner for input handling

## How to Run
1. Make sure you have **Java JDK** installed.
2. Open the terminal or command prompt.
3. Navigate to the folder containing `BankingApp.java`.
4. Compile the Java program:
   ```bash
   javac BankingApp.java
